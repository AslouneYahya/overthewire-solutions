# OverTheWire: Bandit Solutions

I'm happy to share a comprehensive collection of write-ups, command logs, and guides for the [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) wargame.

## Content Structure
- Levels 0-10: Basic commands and file manipulation.
- Levels 11-20: Compression, networking, and permissions.
- Levels 21-34: Cron jobs, shell scripting, and Git.

## Tools Used
- Linux Command Line (Bash)
- OpenSSH
- Git
